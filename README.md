# ShopSphere

A full-stack e-commerce platform built with React, Node.js, Express, Prisma ORM, PostgreSQL, and Razorpay.

## Tech Stack
- **Frontend:** React + Vite, React Router, Axios, Context API
- **Backend:** Node.js, Express.js, JWT Auth, RBAC
- **Database:** PostgreSQL + Prisma ORM
- **Payments:** Razorpay
